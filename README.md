# Aviary
A sample application of Ruby on Rails with MongoDB.

## Dependencies
- Ruby version - 2.2.3
- Rails version - 4.2.4
- Mongoid version - 5.1.3
- MongoDB

## Setup & running the application
- Clone the repo

- Install dependency
```ruby
bundle install
```

- Run the specs
```ruby
rspec spec
```

- Run the server
```ruby
rails s
```
